# Modultester

Dette er en enkel statisk quizportal for TechnoCamp 2026.

## Hva den inneholder

- én treningsdel per modul
- korte oppsummeringer
- multiple choice-spørsmål
- lokal lagring av fremdrift i nettleseren

## Hvordan bruke den

Åpne [index.html](/Users/steinar.stalsberg/Library/CloudStorage/OneDrive-Atea/Projects/TechnoCamp2026/TechnoCamp26_CopilotAgenter/modultester/index.html) i nettleseren, eller publiser mappen som statisk innhold via GitHub Pages.

## Videre utvidelser

- legge til egne scenariooppgaver per modul
- legge til ferdighetsniva eller bestatt-grense
- eksportere resultat lokalt
- splitte ut én underside per modul
